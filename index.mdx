## Why whecho?

whecho is a simple command-line tool that sends webhook notifications when your long-running tasks complete. Perfect for DevOps, data science, and system administration workflows.

## Quick Example

```bash
# Install whecho
pip install whecho

# Configure your webhook
whecho --init

# Send a notification
whecho "Build completed successfully!"
```

## Get Started

<Steps>
  <Step title="Install whecho">
    Install whecho using pip:
    ```bash
    pip install whecho
    ```
  </Step>
  
  <Step title="Get a webhook URL">
    Obtain a webhook URL from your platform of choice (Discord, Slack, Webex, or Teams).
    See our [integrations guide](/integrations/discord) for details.
  </Step>
  
  <Step title="Configure whecho">
    Initialize whecho with your webhook URL:
    ```bash
    whecho --init
    ```
  </Step>
  
  <Step title="Send your first notification">
    Send a test message:
    ```bash
    whecho "Hello from whecho!"
    ```
  </Step>
</Steps>
